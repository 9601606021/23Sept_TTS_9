20. What is the difference between && and || in JavaScript?

|| means “or”. It evaluates to true if the expression on either side is truthy. && means “and”.

Example :

                var first  = 123;
                var second = false;
                var third  = 456;
                var fourth = "abc";
                var fifth  = true;
                alert(first & second); // 0
                alert(first & third);  // 72
                alert(first & fourth); // 0
                alert(first & fifth);  // 1
        
                alert(first && second); // false
                alert(first && third);  // 456
                alert(first && fourth); // abc
                alert(first && fifth);  // true
