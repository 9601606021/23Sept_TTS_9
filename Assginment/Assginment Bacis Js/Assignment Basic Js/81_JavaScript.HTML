Q.81 How can a page be forced to load another page in JavaScript?

In JavaScript, you can force a page to load another page using the window.location object. The window.location object provides information about the current URL and allows you to navigate to a new URL. Here's an example of how you can force a page to load another page using JavaScript:

// Force the page to load a new URL
window.location.href = "https://www.example.com"
