Q.77 How can you convert the string of any base to an integer in JavaScript?

Program :

    
In JavaScript, you can use the parseInt function to convert a string representation of a
        number from any base to an integer. The parseInt function takes two arguments: the string to
        be converted and the base of the numeral system.

        <script>
        let binaryString = "101010";
        let decimalNumber = parseInt(binaryString, 2);
        document.write(decimalNumber); // Output: 42
        </script>
