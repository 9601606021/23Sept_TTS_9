Q.83 Create password field with show hide functionalities

Program :

                <script>
                function check() {
                    var x = document.getElementById("pass");
                    if (x.type === "password") {
                        x.type = "text";
                    }
                    else {
                        x.type = "password";
                    }
                }         
                </script>

Output :


Show Password
