Q.73 What is === operator?

Program :

    
JavaScript Strict Equality Operator is used to compare two operands and return true if both
        the value and type of operands are the same. Since type conversion is not done, so even if
        the value stored in operands is the same but their type is different the operation will
        return false. 

    
    <script>
    var a1 = 2, b1 = 2, c1 = 3;
    var d1 = { name: "Ram" };
    var e1 = { name: "Ram" };
    var f1 = e1;

    document.write("\n a === b :", a1 === b1);
    document.write("\n a === c :", a1 === c1);
    document.write("\n d === e :", d1 === e1);
    document.write("\n f === e :", f1 === e1);
    </script>
