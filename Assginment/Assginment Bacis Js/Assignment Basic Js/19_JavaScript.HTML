19. What is the use of is Nan function?

In JavaScript NaN is short for "Not-a-Number".

The isNaN() method returns true if a value is NaN.

The isNaN() method converts the value to a number before testing it.

Example :

                "Is '9999' NaN? " + isNaN('9999') + "<br>" +
                "Is 'Nixit' NaN? " + isNaN('Nixit') + "<br>" +
                "Is '24/01/2024' NaN? " + isNaN('24/01/2024');
                document.getElementById("ans").innerHTML = result;

            </script>
