Q.74 How can the style/class of an element be changed?

You can modify any CSS property using the style property in a similar way. Keep in mind that this approach modifies the inline style of the element directly.

paragraph.style.color = "blue"; changes the text color to blue.
        

        paragraph.style.fontSize = "20px"; sets the font size to 20 pixels.
        

        paragraph.style.backgroundColor = "lightgray"; changes the background color to light gray.

To change the class of an element in JavaScript, you can use the classList property. The classList property provides methods like add, remove, toggle, and contains to manipulate the classes of an element. Here's an example:

classList.add("newStyle") adds the class "newStyle" to the element.
        

        classList.remove("original") removes the class "original" from the element.
        

        classList.toggle("className") if you want to toggle a class. If the class is present, it will be removed; otherwise, it will be added.
