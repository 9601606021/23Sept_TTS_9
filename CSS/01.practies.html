1.Universal

Test

Universal Selector * { }

The universal selector in CSS is the asterisk (*). It's used to select all elements on an HTML page, including elements inside other elements.

The universal selector is usually written as an asterisk followed by a selector. For example, you can use the universal selector to change the color of all the text on a page to red
